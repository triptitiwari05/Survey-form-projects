# Survey-form-project

A responsive survey form developed using HTML and CSS. This project demonstrates the use of various HTML form elements and responsive styling to create a clean and user-friendly interface.

Features

- User-friendly survey form interface
- Input fields for name, email, age, and contact number
- Dropdown menu for enquiry selection
- Checkbox options for product selection
- Responsive layout for different screen sizes
- Simple and clean design

Technologies Used

- HTML5
- CSS3

Project Structure

survey-form-html-css/
│── index.html
│── style.css
│── README.md

Learning Outcomes

- Creating structured HTML forms
- Styling web pages using CSS
- Working with input fields, dropdowns, and checkboxes
- Designing responsive layouts
- Improving front-end development skills

Future Improvements

- Add JavaScript form validation
- Store submitted responses using a database
- Improve accessibility and user experience
- Enhance responsiveness for all devices
